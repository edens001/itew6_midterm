# CCS Profiling System

A comprehensive profiling system for College of Computer Studies (CCS) that manages students, faculty, courses, schedules, and events.

## Features

### Admin Module
- Student Management (CRUD, Import/Export)
- Faculty Management
- Course Management
- Scheduling System
- Syllabus and Lesson Management
- Event Management
- Comprehensive Search
- Reports Generation

### Faculty Module
- View Schedule
- View Assigned Students
- View Classes
- Profile Management

### Student Module
- Student Registration
- View Schedule
- View Grades
- Profile Management

## Technology Stack

### Frontend
- Vue.js 3
- Vue Router
- Vuex
- Bootstrap 5
- Bootstrap Icons
- SweetAlert2
- Axios

### Backend
- PHP 8.0+
- MySQL
- JWT Authentication
- RESTful API

## Installation

### Prerequisites
- PHP 8.0 or higher
- MySQL 5.7 or higher
- Node.js 14 or higher
- Composer
- Web Server (Apache/Nginx)
